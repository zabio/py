# wxRobot 微信自动回复机器人

### 接入方式
    1.安装python
    2.安装pip
    3.pip install itchat
    
### 使用
    1.切换到项目目录执行 python robot_smart.py
    2.扫码登录

### 自动回复
    与好友聊天自动回复

### 群聊回复
    在微信群里 @自己 后将会自动回复 

### 防撤回功能    
    1.聊天图片默认保存    
    2.撤回后保存在Revocation文件夹内
    3.将撤回消息发送给文件助手作为提示
